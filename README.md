Структура репозитория
/app                  # Backend (например, Laravel/Express)
/client             # Frontend (React/Vue/Vanilla JS)
/database           # Миграции и сиды
/tests              # Автотесты
PROMPTS.md          # Запросы к AI с датой и временем
DECISIONS.md        # Ключевые решения (5–7 пунктов)
README.md           # Инструкция по запуску и тестированию
docker-compose.yml  # Конфигурация Docker
.env.example       # Шаблон переменных окружения

Выбрать стек технологий:

Backend: Node.js + Express или PHP + Laravel (для простоты — Express).

Frontend: React или Vanilla JS (для скорости — Vanilla JS).

БД: SQLite (для локального запуска без доп. настроек).

Docker: Docker Compose.
